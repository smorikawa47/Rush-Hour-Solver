# Rush-Hour-Solver

Given a board file for example:
```
AA...O
.....O
XX...O
...QQQ
....CC
..RRR.
```

Rush Hour Solver outputs its solution like:
```
XR1
XR1
XR1
QL1
OD1
CL1
OD1
OD1
XR1
```

Rush Hour Solver solves **the most difficul**t board:
```
OPPPAA
OBCC.Q
OBXX.Q
RRRD.Q
..EDFF
GGE...
```
In **0.69 seconds**
