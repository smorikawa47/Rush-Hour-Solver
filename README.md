# Rush Hour Solver

## About
Given a board file for example:
```
AA...O
.....O
XX...O
...QQQ
....CC
..RRR.
```

Rush Hour Solver outputs its solution like:
```
XR1
XR1
XR1
QL1
OD1
CL1
OD1
OD1
XR1
```
## Result
The program solves **the most difficult** board (D35.txt):
```
OPPPAA
OBCC.Q
OBXX.Q
RRRD.Q
..EDFF
GGE...
```
In **0.69** seconds
